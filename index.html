pixbit!

features roadmap FAQ github repo

draw pixels. create art.

a beautiful, neo-retro pixel art editor in your browser.

FEATURES

1. BRUSH

the basic drawing tool, duh :p
you can also select the color to draw with!
one pixel at a time.

2. ERASER

delete pixels with this awesome totally unique feature never seen before (kidding)
no, you can't select color. dummy
also one pixel at a time.

3. UNDO

accidental brush stroke? undo!
ctrl+z or click the undo button!

4. REDO

accidental undo? redo!
ctrl+y or click the redo button

5. SAVE TRANSPARENT

save your cool pixel art drawing without a background!

6. SAVE OPAQUE

same thing as transparent but with a white background.
